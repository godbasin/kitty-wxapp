# 吸猫小程序

## 介绍
该小程序是小程序云开发吃鸡的测试小程序，主要用来吸吸猫，顺便帮忙测测小程序云开发的一些 BUG。主要包括模块：
- 别人的猫图
- 我的猫图
- 上传猫图
- 我们的猫图
